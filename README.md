# 🌾 AgriiLens — Hybrid AI-Powered Agricultural Decision Support System

AgriiLens is a state-of-the-art mobile solution designed to transform raw visual data from the field into actionable agricultural intelligence. [cite_start]By integrating deep learning for disease classification and Large Language Models (LLM) for expert advisory, we empower farmers to protect their crops with scientific precision. [cite: 20, 23, 44]

## 🎯 Project Overview
[cite_start]AgriiLens focuses on strategically important crops—**Tomato, Wheat, and Maize**—analyzing plant pathology through images captured in real-world field conditions. [cite: 43] [cite_start]Our goal is to bridge the gap between complex AI diagnostics and practical, human-readable treatment strategies. [cite: 47, 50]

### Key Features
* [cite_start]**Precision Diagnosis:** Leveraging a CNN-based architecture (EfficientNetB0) to achieve over 90% classification accuracy. [cite: 45, 250]
* [cite_start]**Explainable AI (XAI):** Context-aware advisory generation using Groq/Llama 3.3 to provide treatment and prevention strategies. [cite: 46, 166]
* [cite_start]**Field-Ready UX:** Optimized mobile interface designed for low-latency performance (under 5 seconds) and ease of use in outdoor conditions. [cite: 51, 248]
* [cite_start]**Historical Tracking:** Offline access to past diagnostic records and treatment history. [cite: 70, 239]

## 🛠️ Technical Stack

* [cite_start]**Backend:** FastAPI (Python 3.12) for high-performance AI orchestration. [cite: 57, 88]
* [cite_start]**AI Models:** * Image Classification: TFLite Optimized EfficientNetB0. [cite: 102]
